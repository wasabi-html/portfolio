# Portfolio - フロントエンド実装・ECサイト運用 & AI活用ディレクション

フロントエンドエンジニア（HTML/CSSコーダー）としてのポートフォリオサイト、および企画書ドキュメントを管理するリポジトリです。

メインは、ECサイト運用に関わるLP改善、フロント実装、業務効率化、UI改善の実績です。  
あわせて、生成AIを活用した架空ブランド企画・LP構成案・キャラクター企画のコンセプトビジュアルも掲載しています。

## 📝 AI活用プロジェクト概要 (AI Projects)

本リポジトリでは、ターゲットのインサイト分析、企画設計、コピー・LP構成案、生成AIを活用したビジュアルディレクションまでを行った架空ブランドの企画書（Markdown）を格納しています。

各プロジェクトでは、企画意図・ターゲット設計・コピー・構成案を自身で設計し、生成AIはビジュアルラフや方向性の検討に活用しています。

* **[スキンケアブランド「澄刻（すみとき）」](sumitoki.md)**
  * ターゲット：28歳・多忙な営業職女性
  * 企画テーマ：引き算のUI設計と、罪悪感を払拭する「免罪符」スキンケア

* **[若年層向けヘアケアブランド「Dear Me」](dear_me.md)**
  * ターゲット：17歳・初めて自分用のケア用品を買う女子高生
  * 企画テーマ：SNS映えする視覚戦略と、姉妹ブランド体系の構築

* **[架空焼肉ブランド「一牛専心」](yakiniku.md)**
  * 企画テーマ：多店舗展開しないなどの「制約」を価値に変換するブランド戦略

* **[架空食品ブランド「お花ケーキシリーズ」](flowercake.md)**
  * 企画テーマ：データ起点の課題解決（夏場の売り上げ減）と、AIを活用したスピーディな商品展開

* **[お米とお酒の擬人化キャラクター企画](rice.md)**
  * 企画テーマ：お米は幅広い層へ、日本酒は成人向けの食文化コンテンツとして親しみやすく伝えるIP展開
